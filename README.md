# Clone do Spotify (mobile)

Um clone do app do spotify.

Desenvolvido para praticar React Native.

<a href="https://github.com/junioranheu/spotify-api" target="_blank">API</a> feita e adequada para as necessidades do projeto.

Este projeto faz parte do meu potfólio pessoal, sem fins lucrativos, a fim de estudo apenas. Todos os direitos reservados à @spotify.

Projeto iniciado em 11 de abril e finalizado em 30 de abril.
## Aprendizados

Foi meu primeiro contato com React Native. De cara quis encarar um desafio grande: clonar o Spotify. Deu bom!
